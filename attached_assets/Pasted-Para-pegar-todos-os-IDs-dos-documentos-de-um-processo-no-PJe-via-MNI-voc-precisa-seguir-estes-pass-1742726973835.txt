Para pegar todos os IDs dos documentos de um processo no PJe via MNI, você precisa seguir estes passos:

1. Primeiro, faça uma consulta ao processo usando a operação `consultarProcesso` do MNI, configurando o parâmetro `incluirDocumentos` como `true`.

2. Depois, analise a resposta SOAP para extrair os IDs dos documentos.

Veja como implementar isso em Python usando a biblioteca zeep:

```python
from zeep import Client
from zeep.helpers import serialize_object

def obter_ids_documentos_processo(num_processo, cpf_consultante, senha_consultante):
    # URL do serviço WSDL do PJe
    url = "https://pjews.tjce.jus.br/pje1grau/intercomunicacao?wsdl"
    
    # Parâmetros da consulta
    params = {
        'idConsultante': cpf_consultante,
        'senhaConsultante': senha_consultante,
        'numeroProcesso': num_processo,
        'incluirDocumentos': True  # Importante: isso faz retornar a lista de documentos
    }
    
    # Criar cliente SOAP
    client = Client(url)
    
    # Executar a consulta
    response = client.service.consultarProcesso(**params)
    
    # Converter resposta para dicionário Python
    data = serialize_object(response)
    
    # Verificar se a consulta foi bem-sucedida
    if not data['sucesso']:
        print(f"Erro na consulta: {data['mensagem']}")
        return []
    
    # Lista para armazenar os IDs dos documentos
    ids_documentos = []
    
    # Extrair IDs dos documentos
    for documento in data['processo']['documento']:
        ids_documentos.append(documento['idDocumento'])
        
        # Se o documento tiver documentos vinculados, também extraímos seus IDs
        if 'documentoVinculado' in documento:
            docs_vinculados = documento['documentoVinculado']
            # Tratar caso de documento único ou lista
            if isinstance(docs_vinculados, list):
                for doc_vinc in docs_vinculados:
                    ids_documentos.append(doc_vinc['idDocumento'])
            else:
                ids_documentos.append(docs_vinculados['idDocumento'])
    
    return ids_documentos

# Exemplo de uso
if __name__ == "__main__":
    num_processo = "0300039-71.2023.8.06.0203"  # Substitua pelo número do processo
    cpf_consultante = "12345678900"  # Substitua pelo seu CPF/CNPJ
    senha_consultante = "sua_senha"  # Substitua pela sua senha
    
    ids = obter_ids_documentos_processo(num_processo, cpf_consultante, senha_consultante)
    
    print(f"Encontrados {len(ids)} documentos no processo")
    for i, doc_id in enumerate(ids, 1):
        print(f"Documento {i}: ID {doc_id}")
```

Observações importantes:

1. Os IDs dos documentos estão no campo `idDocumento` da resposta XML.

2. Documentos podem ter documentos vinculados (como anexos), que também têm seus próprios IDs.

3. O código acima extrai tanto os IDs dos documentos principais quanto dos documentos vinculados.

4. Se você precisar de mais informações sobre cada documento (como tipo, nome, data), basta extrair esses campos junto com o ID.

5. Certifique-se de ter permissão para acessar o processo, caso contrário a consulta retornará erro.

Esta abordagem funciona para qualquer tribunal que use o PJe com o MNI implementado, incluindo o TJCE.
